# Tripbox

> A tool for organizing and sharing trip confirmations, receipts, emails, etc.

## Features

* Colaborate with other trip members
* Manage edit access (travellers vs observers)
* Custom trip inbox for forwarding trip related emails
* Offline access
* Add to home screen on mobile devices
